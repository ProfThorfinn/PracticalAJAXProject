# 📚 Library Management System (Backend)

A robust backend for a Library Management System built with **Node.js**, **Express**, and **MongoDB Atlas**. This project demonstrates core backend functionalities including RESTful APIs, JWT authentication, and cloud database integration.

---

## 🚀 Features

* **User Authentication:** Secure Login and Registration using `bcrypt` for password hashing and `JWT` for session management.
* **Book Management:** Full CRUD operations (Create, Read, Update, Delete) for managing library inventory.
* **Borrowing Logic:** Functionality for users to borrow and return books with real-time status updates.
* **Cloud Integration:** Fully connected to **MongoDB Atlas** for reliable, cloud-based data storage.
* **Data Seeding:** Custom seeder script to populate the database with realistic book data.

---

## 🛠️ Tech Stack

* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB (Atlas Cloud)
* **ODM:** Mongoose
* **Security:** JSON Web Tokens (JWT) & Bcrypt
